Installation Guide

MAKE SURE YOU OWN A GENUINE COPY OF CUPHEAD AND ARE ON THE LATEST VERSION, 1.3.4

1. Go to the Gamebanana Page and navigate to the Files section. There, download the .rar file and unzip it.

2. Find where your Cuphead is installed. There are many guides on how to do this, and as Cuphead is on GOG and Steam I will not be showing it here.

3. In the Managed folder of the extracted rar file, relocate the "Assembly-CSharp" file to Cuphead_Data/Managed.

4. In the StreamingAssets folder of the extracted rar file, reloace the "devilishcheats" file to Cuphead_Data/StreamingAssets.

And you're done!!
